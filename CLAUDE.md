# CLAUDE.md — QuocDuyApp Plugin

You are working on the **QuocDuyApp** plugin for Claude Code. This plugin provides skills for dependency management, code refactoring, testing, and documentation across multiple languages.

## Available Skills

Invoke via `/qd-<skill-name>` or auto-triggered by keywords:

- `/qd-outdated` — Check outdated dependencies across any language/project
- `/qd-update` — Execute safe, one-by-one dependency updates

## Supported Languages

| Language | Manifest | Package Manager |
|----------|----------|----------------|
| Node.js / React | `package.json` | yarn, npm, pnpm |
| PHP | `composer.json` | Composer |
| Python | `requirements.txt`, `pyproject.toml` | pip, Poetry |
| Go | `go.mod` | go |
| Rust | `Cargo.toml` | cargo |
| Java (Maven) | `pom.xml` | mvn |
| Java (Gradle) | `build.gradle` | gradle |
| C# / .NET | `*.csproj` | dotnet |
| Ruby | `Gemfile` | bundler |
| React Native | `package.json` | yarn, npm |

## Core Principles

- **Never update all libraries at once** — one at a time
- **Always classify risk** before updating (Critical/Minor/Major)
- **Always create baseline** before any update (snapshot, test, branch)
- **Always verify after update** (UI, logic, bundle size)
- **Use dedicated branches** for major version migrations

## Skill Trigger Keywords

| Skill | Triggers |
|-------|----------|
| `/qd-outdated` | "check outdated", "dependency audit", "library versions" |
| `/qd-update` | "update react", "upgrade package", "safe update" |

## Output Files

- `DEPENDENCY-REPORT.md` — Generated by `/qd-outdated`
- `{package}-UPDATE.md` — Generated per update by `/qd-update`

## Rollback

```bash
git revert HEAD
git checkout yarn.lock

```
